## Supported Platforms
Audio Player is known to run on the following host platforms:

- Windows (32/64 bit)
- OS X (also known as macOS)
- Linux (x86/x86_64)

## Build from source
First install npm packages

```bash
npm install
```

Next install

```bash
npm install -g electron-packager asar
```


Buids commands:

For window : 
```bash 
npm run win
```
For linux : 
```bash
npm run linux
```
For OS X also known as macOS :
```bash
npm run macOS
```
